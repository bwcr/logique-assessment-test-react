# Shop Explorer - E-commerce Product Browser

A React application for browsing products, viewing details, and managing a shopping cart using the Platzi Fake Store API.

## Features

- Product browsing with search and category filters
- Product detail pages with image galleries
- Shopping cart with persistent state
- Optimistic UI updates

## Tech Stack

- React 19 + TypeScript
- React Router 7
- TanStack Query for data fetching
- Flowbite + Tailwind CSS
- Vite

## Setup

```bash
pnpm install
pnpm dev
```
